# Mochi Manager

CLI and Python library for Mochi Cards flashcards.

## Quick reference

```bash
# List decks
python mochi_api.py decks

# List cards
python mochi_api.py cards

# Add card
python mochi_api.py add-card "front" "back"

# Add deck
python mochi_api.py add-deck "Deck Name"
```
